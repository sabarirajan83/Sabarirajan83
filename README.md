<summary><b>🔹 QuickFix — Hostel Maintenance Ticket Tracker</b></summary>
<br/>

A **full-stack MERN web application** that digitizes hostel maintenance management end-to-end. Residents raise maintenance tickets, staff resolve them through a structured workflow, and administrators oversee operations with real-time analytics, notification pipelines, and campus-wide announcements.

| Category | Details |
|---|---|
| **Stack** | React.js · Node.js · Express.js · MongoDB · JWT · Tailwind CSS · Azure |
| **Scale** | Multi-role platform (Resident / Staff / Admin) with ticket lifecycle management |
| **Performance** | Optimized REST APIs with indexed MongoDB queries for fast ticket retrieval |
| **Security** | JWT-based authentication with role-based access control (RBAC) |
| **Impact** | Digitized manual maintenance workflows, reducing resolution turnaround time |
| **Repository** | [🔗 GitHub](https://github.com/sabarirajan83/Quickfix-app) · [🌐 Live Demo](https://yellow-moss-0f7d27800.4.azurestaticapps.net) |

This project demonstrates my ability to design **role-based full-stack systems**, implement **secure authentication flows**, and deploy production applications on **Azure Static Web Apps**, while maintaining clean separation between client, server, and data layers.

</details>
